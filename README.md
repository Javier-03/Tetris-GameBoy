Tetris - Game Boy
============
[![Python version](https://img.shields.io/badge/python-3.10.2-blue.svg)](https://www.python.org)
[![Pygame version](https://img.shields.io/badge/pygame-2.1.2-red.svg)](http://pygame.org)

## How to play
| Command      | Key           |
|--------------|---------------|
| Move left    | Left          |
| Move right   | Right         |
| Rotate right | Up /          |
| Soft drop    | Down arrow    |
